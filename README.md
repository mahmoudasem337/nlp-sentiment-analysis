# nlp-sentiment-analysis

This project is a simple and interactive **Sentiment Analysis** web application built using **Python**, **Streamlit**, **TextBlob**, and **Pandas**. The app allows users to upload text or CSV files and get sentiment classifications‚Äîpositive, neutral, or negative‚Äîfor each line of text.

---

##  Features

- Upload `.txt` or `.csv` files for analysis
- Automatically classifies each sentence as:
  - **Positive**
  - **Neutral**
  - **Negative**
- Displays:
  - Count of each sentiment type
  - Detailed sentiment analysis in a table
